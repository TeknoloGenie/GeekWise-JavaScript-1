Selectors

Selectors in javascript are used just like they are in css.
# (hashtag) is used to select an element by ID's
. (dot) is used to select an element by class's
Just using the element name itself for example div or ul selects all those elements. And don't forget you can combine selectors to select a more defined element.
Skill Builder - Selecting Elements

The DOM (Nodes) & Selecting Elements

What is the DOM? Picture it as a Javascript version of your HTML tags.

Selecting Elements on a page

Javascript has a built in function to the document object called querySelector, with it you can select a desired element using a selector query. 
    
<!doctype html>
<html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Website</title>
    </head>
    <body>

        <h1 id="myHdr">This is an h1 tag</h1>

    </body>
</html>
Go to this JSFiddle Page open and your console running type h1. (notice the period). The console should now show you all the various properties and functions available as part of that element.
Use the h1.hasAttribute() function to see if the given dom node has a specific attribute. The hasAttribute takes a string parameter, for example h1.hasAttribute('attribute')
What about adding new attributes? Using the same DOM node we can call the h1.setAttribute(). It takes two parameters, the property name to set, and it's value. Below is an example of how you would add a class attribute to an element. 
    
h1.setAttribute("class", "myClassName");
Not only can can we set attributes, but we can remove them as well. 
    
h1.removeAttribute("class");

Creating Elements

Open up this JSFiddle page and knock out these TODO's as we go over them.
To create an element use the document.createElement() function. It takes one parameter, a string of the element you want to create. So if you wanted to create an anchor tag, you would use document.createElement('a'). Lets make a new element now in our JSFiddle... 
    
var elem = document.createElement('tagName');
All we have is an empty element tag lets assign some other attributes. How about some text (so we have something to see). We can set the text content using the textContent property. 
    
elem.textContent = 'String of text content between the opening and closing tags';
Whats the point in creating an element with some text in it if we dont know how to add it to the page. Thats where the Node.appendChild() comes into play.
Lets just add this anchor tag to the body, we can target the body with document.body. 
    
        document.body.appendChild(elem);
As you can see above, the document.body is a reference to the <body> tag. So it's telling javascript to append a child node to the body tag. This can be done on ANY node you have selected.
Now that we have our element in the body wouldn't it be cool if we could change the text color? 
    
elem.setAttribute('style', 'color: #FF0000');
Some properties of an element can be accessed using the dot notation. One happens to be the href attribute. To get a more specific list check out the Mozilla Developer Network (MDN) - Anchor interface section.
Lets set the color of the text to red using the style attribute of the element tag using the dot notation syntax. 
    
elem.style.color = '#FF0000';
