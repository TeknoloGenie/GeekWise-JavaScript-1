Operators

Not only Mathematical operators, there are Comparison operators as well. Bellow is a list of operators used in JavaScript.
Operators allow us to do something with the variables we've created.

Mathematical

  • = (equals, assignment)
  • + (addition)
  • - (subtraction)
  • * (multiplication)
  • / (division)
  • % (remainder aka modulus)

Shorthand Operators

  • +=
  • -=
  • *=
  • /=

Comparison

  • == (loose comparison)
  • === (strict comparison)
  • != (loose not equal comparison)
  • !== (strict not equal comparison)
  • < (less than)
  • > (greater than)
  • <= (less than or equal to)
  • >= (greater than or equal to)

Logical

  • && (and)
  • || (or)

Math

You can perform mathematical calculations using operators as discussed above.
Try it: Open your console and run a few calculations like 2 + 7, etc...
The "value" of the expression can be stored for later use.
Try it: create a variable called result. Add two numbers together, and store the value in your result variable using the = operator. Log that value to the console.
Shorthand notation exists for quickly operating on an existing variable.
  • +=
  • -=
  • *=
  • /=
With these shortcut helpers, you can turn this: 
    
        var b = 10;
        b = b / 5; //2
into this: 
    
        var b = 10;
        b /= 5;
Operator Precedence, works like it did when you were in sixth grade.
  1. Parentheses
  2. Multiplication / Division
  3. Addition / Subtraction
    
        //what's the answer?
        var a = 5 + 5 * 10;
If you wanted to have the 5 * 10 evaluate first, you would need to use parentheses to group the parts you want to take precedence. 
    
        //what's the answer?
        var a = 5 + (5 * 10);
The remainder % (aka modulus) is a useful operator to know how to leverage.
You can use remainder to determine if something is even or odd, a typical use is if you are generating content like rows in a table, and you want to put an "even" class on even rows and and "odd" class on odd rows.
    
        var a = 5;
        var remainder = a % 2; //1

        var b = 6;
        var remainder = b % 3; //0
It's also very easy to increment / decrement a number by one using javascript.
Use the ++ or -- operators on any existing variable.
    
        var a = 5;
        a++; //6

        var b = 6;
        b--; //5

How can we take a string that we know is a number and convert it to a number datatype? parseInt(number)can be used for turning a string into an whole value integer. parseInt does not round to the nearest whole number, it simply takes everything off after the decimal point. What if you wanted to keep the float value? parseFloat(number). 
    
        var c = "5.4";
        var d = parseInt(c); // number data type 5
        var e = parseFloat(c); // number data type 5.4
If you try and parse a value that cannot be turned into a number, you will get NaN ( not a number)
    
        var a = "foo";
        var b = parseInt(a); // NaN
You can check if a value is NaN by using the isNaN function. 
    
        var a = "foo";
        var b = parseInt(a); //NaN
        isNaN(b); //true
