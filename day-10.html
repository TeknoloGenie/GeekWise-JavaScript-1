Removing items from an array

So now we can create Arrays, access array properties, and add items to arrays, how about removing items?
Items can be removed one at a time from the end of the array, using the pop method. NOTE: This modifies the original array
Try it: Create a variable called lastFriend and assign it to friends.pop(). Then log the variable and what do you get? What about logging the friends array? Has it changed?
Another handy array function we'll talk about is the slice function. It works on arrays a little differently.
The slice function will make a copy of the specified indexes and return them in a new array.
The signature of the slice function looks like: friends.slice(begin, end)
The begin is the starting index, and the end is the index to stop at. NOTE: If the end (param) is not passed it will get everything until the last one.
Try it: Create an array with 4 strings, then slice the array to return the middle 2 strings.
One point to remember about the slice functions is that is does not modify the original array.
If we did want to modify the original array we would use splice! The splice function works in a similar way to the slice function, however it allows you to modify the original array. So imagine this like a cake, where as we are splicing out those pieces of cake and the will never be there again.
The signature of the splice function looks like: 
	
strings.splice(begin, deleteCount, [additional, items, to, add]);
So lets break this apart, the begin param is our starting index, deleteCount is how many items to remove from the starting index, and additional items to add are more elements that you want to add to the array (comma separated values).
Try it: Create an array with at least 6 strings in it. And splice the array to where it leaves the first 2 strings and the last string and add one leaving a total of 4 items in the array (the 3rd item being the new one you added)
We can do a lot more with array's, just look at all it's available functions on the MDN site.

Control Flow - Looping

Looping is the act of repeating a block of code based on a given condition. When the conditions is no longer truthy, the loop stops.
The for loop, maybe the most common loop we'll use in class.
With for loops we need to set up 3 things, our starting point, our condition, and our incrementer.
This type of loop is nice because you do not need to manage the incrementer in your normal code excecution, because it's built into the loop structure. 
	
for (var i = 0; i < 10; i++) {
    console.log(i);
}
What if I want a loop to stop early? Or maybe completely end the loop entirely and all remaining loops?
Javascript provides two ways of controlling the flow of a loop, the continue and break keywords.
continue will immediately reset the loop to the next iteration, and continue on with the loop. It's basically just saying, disregard the rest of this loop and go to the next loop.
Try it: Create a for loop that will loop 10 times, and skip the 5th and 7th loop. All other loops should log the loop number to the console. 
	

for (var i = 0; i < 10; i++) {
    if (i === 5 || i === 7) {
        i++;
        continue;
    }
    console.log(i);
    i++;
}
The break keyword will, when encountered, immediately stop the loop AND all subsequent loops.
Try it: Create a for loop that will loop ten times, each time logging the current loop number, but terminate the loop on the seventh loop. 
	
for (var i = 0; i < 10; i++) {
    if (i === 6) {
        break;
    }
    console.log(i);
}

Looping & Arrays, a perfect match

So all our work with arrays up to this point has been retrieving array indexes one at a time using a numbered bracket notation, for example friends[3]. This is useful, but what if your application requirements were such that you needed to do something to every element in the array?
A for loop is a perfect mechanism for iterating over each element in an array. Lets think about how we can do this. When we write a for loops condition we manually give it an amount of times we want to loop some code. What property in an array tells us how many items are in it?
Try it: Using what we have discussed give it a shot and try to create your first for loop on your own.
Okay, now that i have made you learn how to loop arrays the hard way, time to show you the easy way to do it.
array.forEach(callback, thisArgs) is a javascript method on every array that executes a given callback on every item. There are a few downfalls to this, we cannot force the loop to start at a desired index, and we cant continue or break the loop.
