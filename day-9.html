Arrays

Arrays are very useful for keeping an ordered list of "things".
Array's don't really care what you put into them, they accept all the data types we've talked about to date, including a few we haven't.
You create a new array using the [...] (brackets) syntax
Items in an array are separated with a , (comma)
Try it: Create an array of friends in your chrome console 
    
        var friends = ["billy", "brandon", "jason", "jeremy"];
Array's have a property called length that allows you to determine how many items it is holding.
Try it: log to the console the length of the friends array you created previously by typing friends.length into your console.
Now that we have some data in our arrays, and we know how many items are in the array, now how do we get it back out?

Accessing Array Contents

Array's are index based, so you can use a numbered index to retrieve a specific value by using the bracket notation and passing which index you want. 
	
friends[3]
Try it: Pick one of your friends, count which index value is has and access it using the bracket notation.

Adding items to an array

Okay, so we can get our data back out of an array once its been defined, but how do we add more things to it?
Javascript Array's provide a method called push that allows for adding items to the end of an array.
Try it: Add a new friend to the end of your friends array using the friends.push(...) method. 
	
        friends.push('franklin');
When you add items to your array, the length property is automatically updated for you.
Like stated above, arrays can contain any datatype as its item. So in theory, an array can also contain other arrays. 
	
        //create an array to hold multiple arrays
        var friendGroups = [];
        var jason = ['jason', 'bestie'];
        var jeremy = ['jeremy', 'family'];
        var brandon = ['brandon', 'friend'];

        //add all the friend variables into the friendGroups array
        friendGroups.push(jason);
        friendGroups.push(jeremy);
        friendGroups.push(brandon);
Now lets attempt to pull out the groups only for each of the three friends in our friendGroups array.
Try it: create a snippet that will create an array to hold multiple arrays (defined above). Then write three console logs that log only the colors of the fruits. 
	
        //we can access our array's based on index
        console.log(friendGroups[0][1]);
        console.log(friendGroups[1][1]);
        console.log(friendGroups[2][1]);
