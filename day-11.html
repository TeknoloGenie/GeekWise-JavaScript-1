More DOM Manipulation

Before we can continue on with the final project we need to learn a few more DOM Manipulation techniques, innerHTML, classList and a few others.
Element.innerHTML allows you to change the html within an element. For example we could take a div and change the html from 
    
<div id="changeMe">
    <p>Im just some plain old text in a paragraph element</p>
</div>
To this 
    
<div id="changeMe">
    <p>Now i have an input element too!</p> <input type="text"/>
</div>
Using the following javascript 
    
var div = document.querySelector('#changeMe');

div.innerHTML = '<p>Now i have an input element too!</p> <input type="text"/>';
One more handy Element property to work with is the classList. By targeting the elements classList you have a veriety of new methods to work with to add remove or even toggle classes on an element. But this specific property is something that you can look into on your own over at the Mozilla Developer Network.
