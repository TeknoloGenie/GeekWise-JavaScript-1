While Loop

Loops are blocks of code that repeat until the condition used for the loop is met.
The while statement creates a loop that executes a specified statement as long as the test condition evaluates to true.
The do while statement creates a loop that executes a specified statement as long as the test condition evaluates to true. 
    
while (condition) {
    statement
}
An examples on how to use a while statement. 
    
var n = 0;
var x = 0;

while (n < 5) {
    n++;
    x += n;
}

Debugging

Part of debugging is asking yourself these 3 questions. 80% of the time, asking yourself these questions will answer your problem.
  1. 1. What makes you say your code isn't working?
  2. 2. What did you expect your code to do and why?
  3. 3. What did your code do instead and how do you know?
